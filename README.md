# instruccciones para ejecutar el proyecto
He desarrollado varios componentes React para representar mi hoja de vida dentro de la aplicación. Luego de crearlos, los importé en la estructura principal del proyecto y los agregué al repositorio mediante commits. Posteriormente, ejecuté la aplicación con: npm run dev
Para comprobar que todo funcionaba correctamente.
Además, agregué un componente llamado StackTecnologias, al cual le di un estilo básico, y realicé los commits correspondientes para guardar estos cambios en el historial.
Ahora la aplicación está corriendo en modo de desarrollo, con los componentes de la hoja de vida visibles, y lista para seguir agregando funcionalidades, estilos o mejoras.
